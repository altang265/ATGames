User

    - PlayerColor (String) (hexcode)
    - totalCash (Integer)
    - netWorth (Integer)
    - List of properties/deeds (Deed[])
    - PlayerToken (String)
    - isJailed (Boolean)
    - isBanker (Boolean)
    - hasGetOutOfJailFreeCard (Boolean)
    - isRetired (Boolean)

Tile
    - Name (String)
    - Type (String)
    - Event (function())
